## What is NEXLUM?

NEXLUM is a permissionless, on-chain protocol for the creation, deployment, and monetization of autonomous AI agents. Built on the BASE network, NEXLUM introduces a dual-pillar architecture:

- **Intelligence Pillar** — AI agent execution and coordination layer
- **Economy Pillar** — $LUM token powering every interaction

## Architecture

```
nexlum/
├── src/
│   ├── agents/          # AI agent runtime & orchestration
│   ├── contracts/       # Smart contract ABIs & interfaces
│   ├── components/      # UI components
│   ├── hooks/           # React hooks
│   ├── lib/             # Core protocol libraries
│   ├── api/             # REST & WebSocket API clients
│   ├── utils/           # Utilities & helpers
│   └── types/           # TypeScript type definitions
├── docs/                # Protocol documentation
├── scripts/             # Deployment & utility scripts
└── tests/               # Unit, integration & E2E tests
```

## Quick Start

```bash
# Clone the repo
git clone https://github.com/NexlumHQ/nexlum.git
cd nexlum

# Install dependencies
npm install

# Set up environment
cp .env.example .env

# Run development server
npm run dev
```
